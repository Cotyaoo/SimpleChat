
Простой мессенджер, написанный в PyCharm
Библиотеки: PyQt, Twisted
ПО: QTDesigner

Чат имеет возможность подключать до 100 клиентов
Сервер асинхронный, пока реализованный на localhost

Для проверки программы:
1. Подключить библиотеку PyQT
2. Подключить библиотеку Twisted
3. Запустить Server.py
4. Подключиться в качестве клиента ClientPart.py
5. В появившемся окне написать login:**Свой никнейм**
